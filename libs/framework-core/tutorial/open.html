Maya open latest scene¶

As our first task, we implement an automatic scene open in Maya upon launch. It will check if there is a previous snapshot published on the task, if not it tries to locate a template scene, based on the task, to load and start from.

Prerequisites¶

  1. A shot created in ftrack, with proper timeline and fps set.

  2. The previous custom location plugin deployed, and configured storage scenario set up (preferable).

  3. A Maya template scene to use when no previous published snapshot file exists, present in project folder @ _TEMPLATES/maya/<task type>_template.mb

Implementation¶

All DCC tools goes into the file custom_commands.py:

mypipeline/projects/framework-maya/source/ftrack_framework_maya/utils/bootstrap.py

 1# :coding: utf-8
 2# :copyright: Copyright (c) 2014-2023 ftrack
 3import logging
 4import os
 5import shutil
 6
 7import maya.cmds as cmds
 8
 9logger = logging.getLogger(__name__)
10
11def scene_open(session, logger):
12    '''Load latest scene, or generate new from template.'''
13    from ftrack_framework_maya.utils import get_save_path
14
15    context_id = os.getenv('FTRACK_CONTEXTID')
16    task = session.query('Task where id={}'.format(context_id)).one()
17    path_snapshot_open = path_snapshot_load = None
18    path_snapshot, message = get_save_path(
19        context_id, session, extension='.mb', temp=True
20    )
21    location = session.pick_location()
22    for version in session.query(
23        'AssetVersion where '
24        'task.id={} order by date descending'.format(
25            task['id'],
26        )
27    ).all():
28        # Find a snapshot
29        component = session.query(
30            'Component where '
31            'name="snapshot" and version.id={}'.format(version['id'])
32        ).first()
33        if component:
34            try:
35                path_snapshot_open = location.get_filesystem_path(component)
36            except ftrack_api.exception.ComponentNotInLocationError as e:
37                cmds.confirmDialog(message=str(e))
38
39    if path_snapshot_open is None:
40        # Copy Maya template scene
41        path_template = os.path.join(
42            location.accessor.prefix,
43            task['project']['name'],
44            '_TEMPLATES',
45            'maya',
46            '{}_template.mb'.format(task['type']['name'].lower()),
47        )
48        if os.path.exists(path_template):
49            logger.info(
50                'Copying Maya template {} to {}'.format(
51                    path_template, path_snapshot
52                )
53            )
54            shutil.copy(path_template, path_snapshot)
55            path_snapshot_load = path_snapshot
56        else:
57            logger.warning(
58                'Maya template not found @ {}!'.format(path_template)
59            )
60    else:
61        # Make a copy in temp
62        logger.info(
63            'Copying most recent snapshot {} to {}'.format(
64                path_snapshot_open, path_snapshot
65            )
66        )
67        shutil.copy(path_snapshot_open, path_snapshot)
68        path_snapshot_load = path_snapshot
69
70    if path_snapshot_load:
71        # Load the scene
72        logger.info('Loading scene: {}'.format(path_snapshot_load))
73        cmds.file(path_snapshot_load, open=True, force=True)
74
75
76def set_task_status(status_name, session, logger, unused_arg=None):
77    '''Change the status of the launched task to *status*'''
78    task = session.query(
79        'Task where id={}'.format(os.environ['FTRACK_CONTEXTID'])
80    ).one()
81    status = session.query('Status where name="{}"'.format(status_name)).one()
82    logger.info(
83        'Changing status of task {} to {}'.format(task['name'], status_name)
84    )
85    task['status'] = status
86    session.commit()

We are not going into detail what the scene_open function does, but it tries to locate a previous published snapshot and if not found - a new one is copied from a template and saved to the temp folder and opened.

Finally, to have this run during Maya startup, we add it to userSetup.py:

mypipeline/framework-maya/resources/scripts/userSetup.py

1def initialise():
