# ftrack After Effects integration

Community owned After Effects integration for ftrack.

## Building

### Preparations

1. Install uv.

2. Create and activate a project-local virtual environment:

```bash
cd projects/framework-aftereffects
uv venv .venv
source .venv/bin/activate
```

4. If any dependent libraries updated, make sure to release them to PyPi prior to building the plugin.

5. Update release notes.

6. Set version in `pyproject.toml` (use semantic versioning, for example `MAJOR.MINOR.PATCH` or prerelease `MAJOR.MINOR.PATCHrcN`).

7. If dependencies updated, update the uv lock file. Remember to properly validate/test the change of dependencies.

```bash
uv lock
```

8. Tag and push to SCM

### CI build

See Monorepo build CI

### Manual build

Run all commands below from `projects/framework-aftereffects`.

1. Build with uv:

```bash
uv build
```

2. Build the UXP artifact:

```bash
uv run python ../../tools/build.py --remove_intermediate_folder build_uxp .
```

3. Build Connect plugin (including the UXP artifact):

```bash
uv run python ../../tools/build.py --remove_intermediate_folder --include_assets dist/*.ccx build_connect_plugin .
```

If the build fails and After Effects is using beta or experimental dependencies published to Test PyPi, use the `--testpypi` flag to build the plugin.

To build from source, not involving PyPi, use the `--from_source` flag.

# Development

## UXP plugin

To enable live development, use Adobe UXP Developer Tool and load the plugin
from the staging folder generated by `build_uxp`.

```bash
uv run python ../../tools/build.py --remove_intermediate_folder build_uxp .
```

Build Ftrack Qt Style:

```bash
uv run --with-requirements ../../tools/requirements-pyside6.txt python ../../tools/build.py build_qt_resources --css_only ../../libs/qt-style
```

Create Adobe UXP extension archive:

```bash
uv run python ../../tools/build.py --remove_intermediate_folder build_uxp .
```

### UXP plugin build

The UXP package is written as:

- `projects/framework-aftereffects/dist/ftrack-framework-aftereffects-<version>.ccx`

## Installing

### Connect plugin

Copy the resulting dist/ftrack-framework-aftereffects-<version> folder to your connect plugin folder.
